['target' mean the ip that you targeting]
TCP SYN SCAN 
nmap -sS target  { you may need to use sudo}
TCP CONNECT SCAN 
nmap -sT target { it'snot fast and don't need sudo }
UDP SCAN 
nmap -sU target 
AGGRESSIVE SCAN 
nmap -A target 
CHECK A SPECIFIC PORT
nmap -p 70,300 target 
CHECK A RANGE OR EXTENT OF PORTS
nmap -p 1 - 1200 target
CHECK ALL PORTS 
nmap -p- target 
DISCOVER SERVICES
nmap -sV target
DISCOVER THE OPERATING SYSTEM
nmap -O target
DISCOVER DEVICES ON THE NETWORK
RUN A SPECIFIC SCRIPT 
nmap --script http-enum target 
VULNERABILITY CHECK 
nmap --script vuln target 
nmap -sn 192.168.1.0/24
PASS THE PING IF THE DEVICE NOT RESPONDING 
nmap -Pn target
RUN A DEFAULT SCRIPT 
nmap -sC target
CONTROLLING SPEED, STEALTH 
nmap -T4 target [ T0 = slower 'stealth'][ T5 = faster 'more noise']
